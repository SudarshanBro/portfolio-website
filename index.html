You May Wonder Why ?

1. Why Front-End ?

It offers a blend of creativity, technical skills, and problem-solving, making it a rewarding and dynamic career path. The field is always changing with new tools, frameworks, and design trends. There's always something new to learn and explore.

2. Why React.js ?

Component-based architecture, Virtual DOM concept to create an in-memory representation of the UI, optimizing updates and improving performance

3. My approach to optimizing website performance ?

Image optimization, Minification of code, Reduction of HTTP requests, Leveraging browser caching, Code splitting and Lazy loading.

4. How I handle cross-browser compatibility issues?

Using CSS prefixes, Employing normalization or reset stylesheets, Understanding common browser bugs and Leveraging developer tools for testing

5. How I stay updated with the latest front-end trends and technologies?

Following tech blogs and news outlets, Attending conferences and workshops, Experimenting with new frameworks and libraries and Contributing to open-source projects
